About

"developed by software. Driven by education."

Truthy/Falsey is a quiz based website that allows for children to supplement their educational needs with a fun environment in which they can test themselves on various topics. An easily usable navigation menu has been set up at the top of the page to allow for adults to intuitively move around the website whilst children can press on links within the landing page in order to explore content. The actual quizzes are set up so that children only need to interface through the press of a button which should give them immediate feedback.
